Session Start Prompt for New Qortal Projects

Copy and use this at the beginning of a new AI session.

```text
Read agents/my-real-app-vision.md first.
Then follow agents/master-workflow.md.
Communicate with me in Estonian, but write project files in English.
Before making major architecture decisions, verify Qortal-specific behavior against the official sources referenced in the workflow files when needed.
If something is unclear in the project vision, ask concise clarifying questions before implementing.
```

Quick Variant

```text
Read agents/my-real-app-vision.md first, then follow agents/master-workflow.md. Communicate with me in Estonian, but write project files in English.
```

How to Use

- create a new vision file for the project, for example:
  `agents/my-real-app-vision.md`
- replace the placeholder file name in the prompt if needed
- paste the prompt into the new session
