Project Vision Template for New Qortal qApps

Purpose

This file is a reusable template the user can fill in before asking an
AI agent to create a new Qortal project. It should be short, concrete,
and implementation-oriented.

Instructions for the User

- Replace placeholder values with the real project details.
- Keep the wording simple and direct.
- Focus on what the app must do first, not every future idea.
