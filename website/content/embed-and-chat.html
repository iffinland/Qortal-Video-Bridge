A key part of this project is reuse. The same published QDN video can be posted into Qortal chat or embedded into a standard HTML page.

This page gives a simple public-facing example of that idea: short text, a playable video block, and a clean path into a dedicated detail page. It is intentionally minimal so the content remains easy to maintain.

Later this same website can evolve into the official public-facing home for the app, while still staying simple enough to publish as a static package into Qortal.
