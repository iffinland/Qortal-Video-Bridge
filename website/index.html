Import YouTube videos into QDN with a local-first workflow.

VideoBox Bridge is built for Qortal creators who want a direct, practical path from web2 video sources into QDN publishing.

Open Gallery See Demo Posts

How It Works

  1. Paste a YouTube URL into the app.
  2. Choose a transcode preset.
  3. Let the local helper prepare the video.
  4. Approve the Qortal publish flow.
  5. Reuse the result in posts, chat, and embeds.
Primary Goal Creator-first video importing
Publishing Model Local helper + QDN publish
Reuse Paths Chat, embed, gallery, showcase
